# 🌐 Projeto Front-End — OxeTech Academy

Este projeto foi desenvolvido como **atividade individual** no módulo de **Front-End** do curso **Desenvolvimento Web Full Stack** da OxeTech Academy.  
O objetivo foi colocar em prática os conhecimentos adquiridos em **HTML**, **CSS** e **JavaScript**, criando uma estrutura organizada e funcional para um site moderno.

---

## 🚀 Tecnologias utilizadas

- **HTML5** → estrutura semântica das páginas  
- **CSS3** → estilização e responsividade  
- **JavaScript** → interatividade e dinamismo  
- **Font Awesome** → biblioteca de ícones para enriquecer a interface  

---

---

## 📌 Destaques
- Estrutura de pastas organizada para facilitar manutenção.  
- Separação clara entre **layout.css** (estrutura geral) e **components.css** (componentes reutilizáveis).  
- Uso de ícones modernos com **Font Awesome** para dar mais identidade visual.
